# 20秒三明治制作游戏 - 完整设计文档

## 🎮 项目概览

**游戏名称**：20-Second Sandwich Rush
**游戏类型**：时间管理/快速反应游戏
**目标平台**：Web浏览器 (HTML5)
**开发引擎**：推荐使用 Godot 4.5 或其他支持HTML5导出的引擎
**开发周期**：3周内完成
**单局时长**：20秒严格限制
**项目类型**：20 Second Game Jam 2025 参赛作品

---

## 🎯 核心理念

在严格的20秒限制内，玩家需要从杂乱的冰箱中挑选食材，制作并打包三明治。游戏的核心挑战在于时间管理的策略性选择 - 玩家可以自由分配在挑选、组装、打包三个阶段的时间，但必须在20秒内完成整个过程。

**核心特色：**
- ⏰ **严格20秒限制**：符合Game Jam要求，提供完整游戏体验
- 🧠 **策略性时间分配**：玩家自主决定各阶段时间，做出取舍
- 🥪 **多样化食材选择**：50+种食材，创造不同营养价值和口感组合
- 📊 **营养结算系统**：游戏结束显示详细营养成分，增加教育价值

---

## 🎮 游戏机制设计

### 核心流程
```
游戏开始 → 20秒倒计时开始 → 自由选择食材 → 组装三明治 → 打包完成 → 营养结算
```

### 三阶段玩法（时间自由分配）

#### 1. 食材选择阶段
- **场景**：打开的冰箱门，显示食材网格
- **操作**：点击选择需要的食材
- **策略性**：
  - 快速选择常见食材（省时间但分数普通）
  - 精心挑选高级食材（耗时但营养更丰富）
  - 寻找隐藏的特殊食材（增加探索乐趣）

#### 2. 三明治组装阶段
- **场景**：组装台，显示砧板和食材
- **操作**：拖拽食材到正确位置进行堆叠
- **规则**：
  - 面包必须在最底部和最顶部
  - 中间层可以自由搭配
  - 某些食材组合有特殊效果

#### 3. 打包完成阶段
- **场景**：包装纸和完成区域
- **操作**：一键打包或简单手势操作
- **时间控制**：该阶段耗时较短，主要用于展示完成效果

## 📊 营养结算系统

### 营养成分计算
每局游戏结束后，系统会自动计算并显示：

#### 总营养信息
- **总卡路里**：所有食材卡路里总和
- **蛋白质总量**：影响"力量"评分
- **碳水化合物总量**：影响"能量"评分
- **脂肪总量**：影响"浓郁度"评分

#### 健康评级系统
- **S级 健康之星**：营养均衡，卡路里适中
- **A级 美味佳肴**：口感丰富，营养较好
- **B级 标准餐食**：基本营养需求满足
- **C级 快餐级别**：高热量，营养单一
- **D级 罪恶美食**：超高热量，需谨慎

#### 特殊效果评分
根据选择的食材特殊效果，计算：
- **口味评分**：酸甜苦辣咸的平衡度
- **创意评分**：食材搭配的创新程度
- **奢华评分**：高级食材的使用比例

## 🎮 游戏体验设计

### 难度平衡
- **新手友好**：基础食材容易找到，操作简单直观
- **高手挑战**：寻找完美食材搭配，追求最优时间和营养组合
- **运气因素**：食材位置每局随机，增加重玩性

### 反馈系统
- **视觉反馈**：
  - 食材选中时的高亮效果
  - 营养值的实时显示
  - 倒计时的颜色变化警告
- **音效反馈**：
  - 选择食材的音效
  - 组装成功的音效
  - 时间紧迫的警告音

### 成就系统
- **速度成就**：15秒内完成、18秒内完成等
- **营养成就**：高蛋白、低脂、均衡营养等
- **创意成就**：使用奇特食材、特殊搭配等
- **完美成就**：全S级评价、特殊效果满值等

## 🏗️ 技术实现

### 推荐技术栈：Godot 4.5

#### 核心系统架构

```gdscript
# 游戏管理器
extends Node

signal game_time_updated(remaining_time: float)
signal game_completed(nutrition_data: Dictionary)
signal ingredient_selected(ingredient: Ingredient)

var game_time: float = 20.0
var is_playing: bool = false
var selected_ingredients: Array[Ingredient] = []

# 食材数据结构
class Ingredient:
    var name: String
    var category: String  # 面包、肉类、蔬菜等
    var calories: int
    var protein: float
    var carbs: float
    var fat: float
    var effects: Array[String]
    var texture: Texture2D
```

#### 关键系统模块

1. **食材管理系统**
   - 50+食材的数据库加载
   - 随机位置生成
   - 碰撞检测和选择逻辑

2. **营养计算系统**
   - 实时营养值累加
   - 健康评级算法
   - 特殊效果计算

3. **UI系统**
   - 倒计时显示
   - 食材选择界面
   - 营养结算界面
   - 成就展示

4. **音效系统**
   - 操作反馈音效
   - 背景音乐
   - 时间警告音效

### 性能优化目标
- **包体大小**：<15MB（包含50种食材素材）
- **加载时间**：<3秒（首次加载）
- **内存占用**：<150MB
- **帧率**：稳定60fps

### HTML5导出配置
- 启用WebGL渲染
- 压缩纹理资源
- 优化音频文件
- 确保跨浏览器兼容性
